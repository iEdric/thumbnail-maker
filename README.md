# NeonCover - AI Video Thumbnail Maker

> "创建引人注目的视频缩略图，让你的内容脱颖而出"
> 专业的视频缩略图设计工具

## 📖 项目简介

NeonCover（霓虹封面）是一个专为视频创作者设计的现代化缩略图制作工具。通过直观的拖拽界面、丰富的文本特效和专业的设计元素，帮助你快速创建吸引眼球的视频封面。采用霓虹风格的设计语言，完美契合现代视频内容的审美需求。

## ✨ 主要功能

### 🎨 核心功能

- **可视化编辑器** - 所见即所得的拖拽式设计界面
- **多图层支持** - 文本图层和图片图层自由组合
- **专业字体库** - Inter、Oswald、Playfair Display、Roboto Mono 四种精选字体
- **霓虹风格设计** - 炫酷的霓虹色调和发光效果
- **高清导出** - 支持1920x1080高清PNG格式导出

### 📝 文本编辑功能

- **丰富字体选择** - 四种专业字体系列，每种包含多个字重
- **颜色自定义** - 预设霓虹色彩 + 自定义颜色选择器
- **描边效果** - 可调节的描边宽度和颜色
- **阴影特效** - 自定义阴影模糊度和颜色
- **大小调节** - 灵活的字体大小调整 (20px-300px)
- **文本变换** - 大写转换选项
- **实时预览** - 即时查看所有效果变化

### 🖼️ 图片处理功能

- **图片上传** - 支持本地图片文件导入
- **尺寸调节** - 保持宽高比的智能缩放 (50px-1000px)
- **边框装饰** - 可自定义的边框宽度和颜色
- **旋转控制** - 360度自由旋转
- **透明度调节** - 0-100%不透明度控制

### 🎭 背景特效

- **背景图片** - 上传自定义背景图片
- **填充模式** - 填充(Fill)和适应(Fit)两种模式
- **滤镜效果** - 亮度、对比度、饱和度、模糊度四项调节
- **覆盖层** - 可调节透明度的彩色覆盖层
- **网格背景** - 空状态下的设计辅助网格

## 🛠️ 技术栈

- **前端框架**: React 19.2.3
- **开发语言**: TypeScript 5.8.2
- **构建工具**: Vite 6.2.0
- **样式方案**: Tailwind CSS (CDN)
- **图标库**: Lucide React
- **字体**: Google Fonts (Inter, Oswald, Playfair Display, Roboto Mono)
- **导出功能**: html-to-image
- **路由**: React Router DOM
- **唯一ID**: Nanoid

## 💡 使用说明

### 1. 开始设计

1. **访问应用** - 在浏览器中打开应用首页
2. **添加文本** - 点击底部的"Text"按钮添加文本图层
3. **添加图片** - 点击底部的"Image"按钮上传图片文件
4. **编辑内容** - 选择图层后在右侧面板中进行属性编辑

### 2. 编辑工作流程

#### 背景设置 (BACKDROP)
- **上传背景** - 点击"Upload Background"上传背景图片
- **填充模式** - 选择Fill(填充)或Fit(适应)显示模式
- **滤镜调节** - 调整亮度、对比度、饱和度、模糊度
- **覆盖层** - 设置覆盖颜色和透明度

#### 图层编辑 (LAYER)
- **选择图层** - 点击画布上的图层或在背景面板的图层列表中选择
- **编辑文本** - 修改文字内容、字体、大小、颜色等
- **编辑图片** - 调整图片尺寸、边框、旋转角度
- **删除图层** - 点击图层右上角的删除按钮

#### 通用属性
- **位置调整** - 在画布上拖拽图层改变位置
- **旋转控制** - 使用旋转滑块调整角度
- **透明度** - 调节图层不透明度
- **阴影效果** - 添加和自定义阴影

### 3. 导出作品

- **预览模式** - 导出前会自动取消选中状态
- **高清导出** - 点击"Export"按钮导出1920x1080 PNG图片
- **自动命名** - 文件名包含时间戳防止覆盖

NeonCover的设计理念：

- **视觉冲击** - 霓虹风格设计，创造吸引眼球的效果

NeonCover是一个纯前端应用，无需服务器支持：
- 所有设计数据都在浏览器本地处理
